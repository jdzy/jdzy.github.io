Hexo + Github免费搭建个人博客

发表于 2021-10-15 | 分类于 Github

1.前言

使用Github page服务搭建博客的好处有：

  • 全是静态文件访问速度快

  • 免费，不用花一分钱

  • Github免费提供二级域名

  • 可轻松打包转发到第三平台，本站采用Vercel关联Github仓库进行托管，国内访问速度也很快

1.1.开发环境

  • Windows 10
  • Node.js@12.14.0
  • Git@2.32.0
  • Hexo-cli@4.3.0
  • Next@5.1.4
